# Resume Project

This repository contains a ready-to-use modern resume built with pure HTML, CSS, and JavaScript. 

## Structure

- `/html_resume/index.html` - The main HTML document containing the resume content.
- `/html_resume/style.css` - Custom styling ensuring a modern, responsive layout.
- `/html_resume/script.js` - JavaScript for interactivity (like the print button and scroll animations).

## How to edit in VS Code

1. Open this project in **Visual Studio Code**.
2. Navigate to the `html_resume` folder.
3. Open `index.html` and update the dummy text with your actual information (Name, Experience, Education, etc.).
4. Use the **Live Server** extension in VS Code to preview your changes in real-time. Right-click `index.html` and select "Open with Live Server".

## Features
- Fully responsive (works perfectly on mobile and desktop).
- Print-friendly styles built in.
- Interactive animations and clean typography.
- Easily customizable color variables in `style.css`.
